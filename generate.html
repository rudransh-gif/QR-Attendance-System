QR Entry System

Admin Panel

Generate QR codes for all registered users

How it works:
1. Open your Google Sheet
2. Use the 🎫 QR System menu
3. Run Setup Sheets first
4. Add users to the Users sheet (ID + Name)
5. Run Generate All QR Codes
6. QR images are saved to your Google Drive → QR_Codes folder
